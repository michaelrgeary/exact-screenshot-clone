# Learnings: Structure Assembler

## Best Practices

### Assembly Order
1. Title page
2. Copyright
3. Table of Contents
4. Front matter (intro, how to use)
5. Chapters in order
6. Back matter (conclusion, resources)
7. Index

### Verification
- All chapters present and numbered
- Cross-references resolve correctly
- Page/chapter references accurate
- Consistent formatting throughout

### Metadata
- Chapter titles match TOC
- Word counts recorded
- Quality scores attached
- Version information included

## Do's
- Verify chapter order
- Check all links/references
- Ensure consistent formatting
- Include all required sections

## Don'ts
- Don't miss any chapters
- Don't break page references
- Don't mix up ordering
- Don't skip verification
