# Learnings: Spanish Proofreader

## Best Practices

### Proofreading Focus
- Grammar and spelling errors
- Accent marks (critical in Spanish)
- Punctuation (different conventions)
- Consistency with style guide

### Common Errors
- Missing accents (él vs. el)
- Gender agreement
- Verb conjugation errors
- Punctuation placement

### Quality Verification
- Reads smoothly
- No English fragments
- Consistent terminology
- Proper formatting

## Do's
- Check every accent mark
- Verify gender agreement
- Read aloud for flow
- Compare to English for completeness

## Don'ts
- Don't change style (that's Naturalizer's job)
- Don't miss accents on capitals (Él)
- Don't introduce new errors
- Don't skip any sections
