# Learnings: Back Matter Writer

## Best Practices

### Required Sections
- Conclusion/Final thoughts
- Quick reference summary
- Additional resources
- About the author
- Call to action (next steps)

### Tone
- Celebratory (reader finished!)
- Forward-looking
- Encouraging next steps
- Invitation to connect

### Content
- Summarize key themes
- Provide resource list
- Include contact information
- Ask for reviews/referrals

## Do's
- Leave reader feeling accomplished
- Provide clear next steps
- Include useful resources
- Make it easy to stay connected

## Don'ts
- Don't introduce new concepts
- Don't end abruptly
- Don't be too sales-y
- Don't forget actionable next steps
