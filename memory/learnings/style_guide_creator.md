# Learnings: Style Guide Creator

## Best Practices

### Voice Analysis
- Is it first person ("I") or second person ("you")?
- Formal or conversational?
- Authoritative or peer-to-peer?
- Humor level (serious, occasional wit, frequently funny)?

### Formatting Conventions
- How are scripts/dialogues formatted?
- Are there consistent callout boxes?
- How are key points emphasized?
- What heading hierarchy is used?

### Chapter Structure Patterns
- Standard opening (story? question? statistic?)
- Section organization
- Closing conventions (summary? call to action?)
- Typical chapter length

## Do's
- Include specific examples for each guideline
- Document the "feel" of the writing
- Note trademark phrases or expressions
- Specify reading level target (Grade 7-8)

## Don'ts
- Don't create rules that conflict with source style
- Don't be so rigid it stifles good writing
- Don't ignore the author's intentional variations
- Don't forget to document exceptions
