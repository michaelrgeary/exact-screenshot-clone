# Learnings: Prep Validator

## Best Practices

### Validation Checks
- All chapters extracted (count matches TOC)
- Each chapter has content (not empty)
- Structure document complete
- Style guide has required sections

### Pass/Fail Criteria
- Chapter count must match expected
- No empty chapters
- Structure covers all themes
- Style guide is actionable

## Do's
- Verify chapter boundaries are correct
- Check for content completeness
- Validate style guide usability
- Log specific issues found

## Don'ts
- Don't pass with missing chapters
- Don't accept empty content
- Don't skip structure verification
- Don't ignore style guide gaps
