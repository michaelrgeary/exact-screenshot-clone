# Learnings: Script Adapter

## Best Practices

### Script Context Conversion
- Phone scripts → Door-to-door scripts
- Email templates → Text/voicemail templates
- Meeting room presentations → Driveway/kitchen table
- Corporate buyer → Homeowner decision-maker

### Roofing-Specific Elements
- Reference inspection findings
- Mention visible damage (missing shingles, etc.)
- Include insurance claim language
- Address common roofing objections

### Natural Language
- Scripts should sound conversational
- Avoid robotic or salesy language
- Include pause points and listening cues
- Allow for personalization

## Do's
- Include stage directions (pause here, show photos)
- Provide variations for different scenarios
- Note where to customize with specific details
- Make scripts pronounceable (read aloud test)

## Don'ts
- Don't create scripts that sound canned
- Don't ignore the homeowner's likely responses
- Don't make scripts too long to memorize
- Don't use jargon the homeowner won't understand
