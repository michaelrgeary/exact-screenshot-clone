# Learnings: Glossary Builder

## Best Practices

### Term Selection
- Include all roofing-specific terms used in the book
- Add common industry abbreviations
- Include sales terms unique to the book
- Note regional variations if applicable

### Definition Quality
- Clear, concise definitions (1-2 sentences)
- Include pronunciation if not obvious
- Add context for when the term is used
- Cross-reference related terms

### Categories
- Roofing materials (shingles, flashing, etc.)
- Roofing processes (tear-off, installation, etc.)
- Sales terminology (closing, upsell, etc.)
- Insurance/claims language
- Measurements and specifications

## Do's
- Keep definitions accessible to newcomers
- Include terms a salesperson needs to sound credible
- Note terms that impress homeowners
- Add examples where helpful

## Don'ts
- Don't include terms not used in the book
- Don't over-complicate definitions
- Don't duplicate existing dictionary definitions
- Don't forget Spanish translations are needed
