# Learnings: Grammar Checker

## Best Practices

### Priority Issues
- Subject-verb agreement
- Tense consistency
- Pronoun clarity
- Comma splices and run-ons

### Style Considerations
- Sentence fragments OK if intentional
- Informal grammar OK in dialogue
- Consistent punctuation choices
- American English spelling

### Common Errors
- Their/there/they're confusion
- Misplaced modifiers
- Dangling participles
- Inconsistent list punctuation

## Do's
- Fix clear errors
- Note stylistic choices (don't change)
- Preserve dialogue authenticity
- Check script formatting separately

## Don'ts
- Don't over-correct conversational style
- Don't change intentional fragments
- Don't make prose feel robotic
- Don't miss basic errors while hunting complex ones
