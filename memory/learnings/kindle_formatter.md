# Learnings: Kindle Formatter

## Best Practices

### Format Requirements
- EPUB 3.0 compatible
- Reflowable text
- Embedded fonts (Caveat for diagrams)
- Proper metadata

### Technical Considerations
- CSS for styling
- NCX for navigation
- OPF for manifest
- Responsive images

### Quality Checks
- Test on multiple devices
- Verify table of contents works
- Check image rendering
- Validate EPUB structure

## Do's
- Use semantic HTML
- Include alt text for images
- Test navigation links
- Optimize file size

## Don'ts
- Don't use fixed layouts unnecessarily
- Don't embed huge images
- Don't break accessibility
- Don't skip validation
