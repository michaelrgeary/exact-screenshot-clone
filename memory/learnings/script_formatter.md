# Learnings: Script Formatter

## Best Practices

### Script Format
- Clear speaker labels (Salesperson: / Homeowner:)
- Dialogue in quotation marks or indented
- Stage directions in brackets [pause here]
- Consistent formatting throughout book

### Script Types
- Full dialogue scripts
- Response templates
- Opening lines
- Objection handlers

### Enhancement
- Add stage directions where helpful
- Include expected responses
- Note variations for different scenarios
- Make scripts scannable

## Do's
- Standardize all script formatting
- Include context before scripts
- Add helpful cues [smile] [pause]
- Make scripts easy to practice

## Don'ts
- Don't over-direct (let salesperson adapt)
- Don't format non-scripts as scripts
- Don't create inconsistent formatting
- Don't add cues that feel robotic
