# Learnings: Edit Validator

## Best Practices

### Validation Checks
- All critical issues resolved
- Grammar score above threshold
- No unresolved term inconsistencies
- Fact check issues addressed

### Quality Thresholds
- No critical issues remaining
- Reading level appropriate (Grade 7-8)
- Consistent terminology
- Flow issues resolved

## Do's
- Verify issue resolution
- Check grammar quality
- Validate consistency
- Confirm reading level

## Don'ts
- Don't pass with critical issues
- Don't accept poor grammar
- Don't skip consistency check
- Don't ignore flow problems
