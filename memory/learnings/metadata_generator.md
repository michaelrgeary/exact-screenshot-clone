# Learnings: Metadata Generator

## Best Practices

### Required Metadata
- Title and subtitle
- Author name(s)
- Publisher information
- Language (en, es)
- Description/synopsis

### Platform-Specific
- Amazon: BISAC categories, keywords
- Apple Books: Subject codes
- General: ISBN, publication date
- All: Series information if applicable

### Keywords/Categories
- Relevant to content
- Searchable terms
- Category-specific requirements
- Not keyword-stuffed

## Do's
- Complete all required fields
- Use platform-appropriate formats
- Include both languages
- Verify accuracy

## Don'ts
- Don't leave fields empty
- Don't use irrelevant keywords
- Don't miss platform requirements
- Don't forget Spanish metadata
