# Learnings: Translation Prep Validator

## Best Practices

### Validation Checks
- All glossary terms have Spanish translations
- Spanish style guide complete
- No missing terminology
- Consistent translation choices

### Quality Criteria
- Translations verified for accuracy
- Style guide is comprehensive
- Regional appropriateness confirmed
- Usage notes included where needed

## Do's
- Verify complete term coverage
- Check translation accuracy
- Validate style guide completeness
- Confirm dialect consistency

## Don'ts
- Don't pass with untranslated terms
- Don't accept incomplete style guide
- Don't skip accuracy verification
- Don't ignore regional concerns
