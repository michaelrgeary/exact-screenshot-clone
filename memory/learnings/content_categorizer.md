# Learnings: Content Categorizer

## Best Practices

### Category Definitions
- **Mindset**: Mental frameworks, attitudes, beliefs
- **Technique**: Specific actions, methods, approaches
- **Script**: Word-for-word dialogues, phrases to use
- **Framework**: Step-by-step processes, systems
- **Story**: Narratives that illustrate concepts
- **Tip**: Quick, actionable advice

### Sales Stage Mapping
- Prospecting (finding leads)
- First contact (initial approach)
- Discovery (understanding needs)
- Presentation (showing value)
- Objection handling
- Closing
- Follow-up
- Referral generation

### Multi-Category Items
- Some content fits multiple categories
- Assign primary and secondary categories
- Note when a technique includes a script
- Link related items

## Do's
- Be consistent in categorization criteria
- Create clear boundaries between categories
- Note confidence level if uncertain
- Cross-reference with similar items

## Don'ts
- Don't force items into categories
- Don't create overlapping category definitions
- Don't miss the author's own categorization
- Don't over-categorize (keep it useful)
