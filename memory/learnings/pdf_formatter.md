# Learnings: PDF Formatter

## Best Practices

### Output Types
- **Screen PDF**: Optimized for reading on devices
- **Print PDF**: Optimized for physical printing
- Both: Proper margins and bleed

### Formatting
- Consistent page layout
- Professional typography
- Proper headers/footers
- Page numbers

### Print Considerations
- Bleed areas for images
- CMYK color space
- High-resolution images (300 DPI)
- Proper trim marks

## Do's
- Use proper page sizes
- Include bookmarks/navigation
- Optimize for intended use
- Verify print quality

## Don'ts
- Don't mix RGB/CMYK
- Don't use too-small fonts
- Don't forget about margins
- Don't skip proofing
