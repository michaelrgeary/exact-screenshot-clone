# Learnings: Index Generator

## Best Practices

### Index Entries
- Key concepts and terms
- Technique names
- Important topics
- Commonly searched terms

### Entry Format
- Alphabetical order
- Page numbers or chapter references
- Sub-entries for broad topics
- Cross-references ("See also...")

### Quality
- Comprehensive but not exhaustive
- Useful for quick reference
- Consistent capitalization
- Accurate references

## Do's
- Include all glossary terms
- Add commonly searched topics
- Use sub-entries for broad topics
- Verify all references

## Don'ts
- Don't index every mention
- Don't miss key concepts
- Don't create confusing entries
- Don't forget ebook vs. print differences
