# Learnings: Transform Validator

## Best Practices

### Validation Checks
- Every tactic has roofing context
- Stories rewritten (not just copied)
- New roofing terms added to glossary
- No orphaned content

### Transformation Quality
- Roofing context feels authentic
- Original meaning preserved
- Terminology used correctly
- Scripts adapted properly

## Do's
- Verify transformation completeness
- Check glossary population
- Validate roofing authenticity
- Compare to source for fidelity

## Don'ts
- Don't pass untransformed content
- Don't accept forced roofing context
- Don't skip glossary verification
- Don't ignore copied content
