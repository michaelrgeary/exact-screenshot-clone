# Learnings: Analysis Validator

## Best Practices

### Validation Checks
- At least 1 tactic extracted per chapter
- All tactics have categories
- Stories have source locations
- Cross-refs point to valid chapters

### Quality Thresholds
- Minimum tactics per chapter varies by source
- All tactics must be categorized
- Stories must have context
- No orphan references

## Do's
- Verify categorization completeness
- Check cross-reference validity
- Validate story extraction quality
- Confirm duplicate detection ran

## Don'ts
- Don't pass with uncategorized tactics
- Don't accept invalid references
- Don't skip duplicate check
- Don't ignore missing analysis
