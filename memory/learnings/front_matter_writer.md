# Learnings: Front Matter Writer

## Best Practices

### Required Sections
- Title page (book title, subtitle, author)
- Copyright page (year, publisher, ISBN if available)
- Dedication (if applicable)
- Introduction/Welcome
- How to Use This Book

### Tone
- Welcoming and encouraging
- Sets expectations
- Matches book voice
- Builds excitement

### Content
- Why this book exists
- Who it's for
- What they'll learn
- How to get the most from it

## Do's
- Make reader feel welcomed
- Set clear expectations
- Provide a "quick start" option
- Include contact/feedback info

## Don'ts
- Don't make intro too long
- Don't repeat chapter content
- Don't be generic
- Don't forget to match book voice
