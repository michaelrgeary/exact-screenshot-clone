# Learnings: Diagram Validator

## Best Practices

### Validation Checks
- Image renders without error
- Content matches specification
- Text is legible
- Diagrams have captions

### Technical Quality
- File size reasonable
- Resolution appropriate
- Format correct (PNG/SVG)
- Colors accessible

## Do's
- Test render each diagram
- Verify text readability
- Check specification match
- Validate caption presence

## Don'ts
- Don't pass unreadable diagrams
- Don't accept broken renders
- Don't skip caption check
- Don't ignore accessibility issues
