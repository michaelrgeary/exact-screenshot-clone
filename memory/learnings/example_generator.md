# Learnings: Example Generator

## Best Practices

### Example Types
- Quick illustration (1-2 sentences)
- Mini-case study (paragraph)
- Before/after comparison
- "Imagine this scenario..."

### Example Placement
- After introducing a concept
- When tactic might be unclear
- To show variations of a technique
- When emotional connection needed

### Roofing Authenticity
- Use realistic roofing scenarios
- Include believable homeowner reactions
- Reference actual materials and processes
- Show real challenges salespeople face

## Do's
- Make examples specific and concrete
- Vary the scenarios (not all storm damage)
- Show what success looks like
- Include failed attempts too (learning moments)

## Don'ts
- Don't use examples that are too similar
- Don't make examples longer than the concept
- Don't include unrealistic scenarios
- Don't over-explain (let example speak)
