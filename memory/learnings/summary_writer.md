# Learnings: Summary Writer

## Best Practices

### Summary Purpose
- Enable cross-references from future chapters
- Provide quick reference for readers
- Capture the essence in 2-3 sentences
- Highlight the actionable takeaways

### Content Selection
- Main teaching point(s)
- Key tactic or technique introduced
- Most memorable example/story
- Primary call to action

### Format
- Brief paragraph (50-100 words)
- Optionally include bullet points for tactics
- Should stand alone (make sense without chapter)
- Uses chapter's terminology

## Do's
- Write for someone who hasn't read the chapter yet
- Include the "so what" - why does this matter?
- Make it quotable and shareable
- Enable other chapters to reference effectively

## Don'ts
- Don't try to include everything
- Don't be vague or generic
- Don't use filler language
- Don't spoil the best stories (tease instead)
