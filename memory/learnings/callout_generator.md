# Learnings: Callout Generator

## Best Practices

### Callout Types
- **Pro Tip**: Insider advice, expert shortcuts
- **Warning**: Common mistakes to avoid
- **Quick Reference**: Summary for easy recall
- **Try This**: Immediate action suggestion
- **Real Talk**: Honest, direct advice

### Content Selection
- Highly quotable statements
- Critical warnings
- Memory aids and mnemonics
- Quick summaries of key concepts
- Actionable tips

### Formatting
- Keep callouts brief (2-3 sentences max)
- Use consistent callout styling
- Include icon/emoji indicators
- Don't overuse (3-5 per chapter)

## Do's
- Make callouts scannable
- Include unique insight not in main text
- Vary callout types for visual interest
- Place callouts at natural break points

## Don'ts
- Don't repeat main text in callout
- Don't make callouts too long
- Don't use more than 5 per chapter
- Don't interrupt critical narrative flow
