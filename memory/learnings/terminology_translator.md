# Learnings: Terminology Translator

## Best Practices

### Translation Quality
- Use neutral Latin American Spanish
- Verify with roofing industry sources
- Include usage notes
- Note regional variations if significant

### Term Categories
- Roofing materials (maintain accuracy)
- Sales terminology (maintain meaning)
- Idiomatic expressions (adapt appropriately)
- Technical specifications (verify carefully)

### Consistency
- One Spanish term per English term
- Document decision if multiple options
- Note when context changes preferred term
- Update glossary entries immediately

## Do's
- Verify with native speakers if unsure
- Include pronunciation notes if helpful
- Note formal vs. informal usage
- Create consistent terminology

## Don'ts
- Don't use Spain-specific Spanish
- Don't translate idioms literally
- Don't miss industry-specific meanings
- Don't create inconsistent mappings
