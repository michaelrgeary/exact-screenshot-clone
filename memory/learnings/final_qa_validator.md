# Learnings: Final QA Validator

## Best Practices

### Validation Checks
- All files generate without error
- Metadata complete for each format
- File sizes reasonable
- Quality scores meet thresholds

### Final Verification
- EPUB validates properly
- PDF renders correctly
- Both languages complete
- All outputs accessible

## Do's
- Test open each file
- Verify metadata completeness
- Check file sizes
- Confirm quality thresholds met

## Don'ts
- Don't pass with corrupted files
- Don't accept incomplete metadata
- Don't skip render testing
- Don't ignore quality failures
