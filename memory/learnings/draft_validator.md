# Learnings: Draft Validator

## Best Practices

### Validation Checks
- Chapter within word count range
- All outline sections written
- Summary exists and correct length
- Cross-references properly formatted
- No placeholder text remaining

### Content Completeness
- Opening hook present
- All sections from outline
- Transitions smooth
- Takeaways included

## Do's
- Verify word count bounds
- Check for placeholder text
- Validate cross-references
- Confirm summary quality

## Don'ts
- Don't pass incomplete drafts
- Don't accept [PLACEHOLDER] text
- Don't skip reference validation
- Don't ignore word count limits
