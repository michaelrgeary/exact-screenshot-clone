# Learnings: TOC Generator

## Best Practices

### TOC Format
- Chapter numbers and titles
- Section headings (optional)
- Page numbers (for PDF)
- Clickable links (for ebook)

### Consistency
- Match exactly to chapter titles
- Consistent capitalization
- Parallel structure
- Appropriate hierarchy

### Presentation
- Clean, scannable layout
- Consistent indentation
- Page numbers aligned
- Professional appearance

## Do's
- Verify against actual chapters
- Include front/back matter entries
- Make links work (test them)
- Match both languages consistently

## Don'ts
- Don't include placeholder titles
- Don't miss any chapters
- Don't use inconsistent formatting
- Don't create broken links
