# Learnings: Translation Validator

## Best Practices

### Validation Checks
- All glossary terms used consistently
- Word count proportional (10-15% longer)
- No untranslated English fragments
- Diagram captions translated

### Quality Criteria
- Natural-sounding Spanish
- Meaning preserved
- Style guide followed
- Terminology consistent

## Do's
- Check term consistency
- Verify length proportions
- Scan for English fragments
- Validate caption translations

## Don'ts
- Don't pass with inconsistent terms
- Don't accept drastically wrong lengths
- Don't miss English fragments
- Don't skip caption check
