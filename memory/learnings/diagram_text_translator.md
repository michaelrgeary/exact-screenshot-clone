# Learnings: Diagram Text Translator

## Best Practices

### Space Constraints
- Spanish text is typically longer
- Keep translations concise
- Abbreviate if necessary
- Maintain readability

### Consistency
- Use glossary terms exactly
- Match caption terminology
- Consistent across all diagrams
- Same abbreviation style

### Visual Considerations
- Text must fit in diagram space
- Consider line breaks
- Maintain visual balance
- Ensure legibility

## Do's
- Translate all text elements
- Check fits in allotted space
- Verify with captions
- Use consistent terminology

## Don'ts
- Don't translate decorative elements
- Don't exceed space limits
- Don't inconsistently abbreviate
- Don't miss any text elements
