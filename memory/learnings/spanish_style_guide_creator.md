# Learnings: Spanish Style Guide Creator

## Best Practices

### Voice Mapping
- Maintain conversational tone
- Use "tú" for reader address
- Use "usted" in customer scripts
- Keep energy level consistent

### Spanish Considerations
- Neutral Latin American dialect
- Avoid regional slang
- Maintain reading level target
- Account for text expansion (10-20%)

### Format Adaptation
- Adjust punctuation conventions
- Handle quotation marks (« » vs. " ")
- Adapt number formats if needed
- Maintain heading structure

## Do's
- Base on English style guide
- Include Spanish-specific examples
- Note differences from English
- Address common translation pitfalls

## Don'ts
- Don't change the book's personality
- Don't use Spain-specific conventions
- Don't ignore cultural differences
- Don't over-formalize
