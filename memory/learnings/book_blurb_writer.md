# Learnings: Book Blurb Writer

## Best Practices

### Blurb Types
- Short description (50-100 words)
- Long description (150-300 words)
- Back cover copy
- Category-specific versions

### Content Structure
- Hook (grab attention)
- Problem (what reader faces)
- Promise (what book delivers)
- Credibility (why trust this)
- Call to action

### Style
- Benefits over features
- Action-oriented language
- Emotional connection
- Clear and punchy

## Do's
- Write multiple lengths
- Focus on reader benefits
- Include social proof if available
- Create versions for both languages

## Don'ts
- Don't be too long
- Don't list every chapter
- Don't use clich√©s
- Don't forget Spanish version
